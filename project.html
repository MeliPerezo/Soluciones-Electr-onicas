Sobre el proyecto

  • Esta startup ofrece realizar consultas arancelarias y acceder a bases de datos para obtener informacion sobre consultas previas.
  • Nos diferenciamos del resto por ser una plataforma sencilla de manejar, es muy intuitiva.
  • Facilita los procesos para muchos profesionales dedicados al comercio.

Consulta arancelaria

El objetivo de este servicio es que usted como usuario pueda hacer consultas sobre impuestos, regulaciones arancelarias, criterios, I.V.A y otros permisos necesarios para importar o exportar un producto.

Esto se realiza gracias a un formulario que el usuario debe completar para esperar una respuesta por parte del equipo.

Acceder a la Base de Datos

El objetivo de la Base de Datos es permitir que cualquier persona pueda acceder y buscar un producto.

Para esto se debe rellenar un pequeño formulario para saber que cuál es el producto que desea buscar, no es necesario rellenarlo por completo.

Una vez que le de click en buscar, abajo de la página encontrará los productos almacenados en la base de datos.
